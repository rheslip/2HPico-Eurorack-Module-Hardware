**4HPico-Eurorack-Module-Hardware

4HP Eurorack module based on Raspberry Pi RP2350 Pico 2

May 6/2026 - updated to V1.1 which corrects missing GND plane connection from middle to bottom of PCB. 
not tested but changes are very minor 

April 13/2026 - added 4HPico KiCad files V1.0. Boards fabbed and tested, working OK with jumper to connect middle and bottom GND planes.

Panel .stl file for 3D printing included. No plans to do a panel PCB at this point.

4HPico DSP is basically the 2HPico DSP with stereo audio in and out and two dedicated CV inputs. Encoder with a 64x32 OLED display instead of pots for setting parameters. This is a single PCB design which requires a 3D printed panel or PCB panel.
