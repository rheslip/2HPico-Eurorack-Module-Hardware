# 4HPico-Eurorack-Module-Hardware

4HP Eurorack module based on Raspberry Pi RP2350 Pico 2

April 13/2026 - added 4HPico KiCad files. Untested as of 4/13/2026. Panel design TBD.

4HPico is basically the 2HPico DSP with stereo audio in and out and two dedicated CV inputs. Encoder with a 64x32 OLED display instead of pots for setting parameters. This is a single PCB design which requires a 3D printed panel or PCB panel.
